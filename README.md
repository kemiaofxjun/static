## 上传命令

### 将更改提交
git add .
git commit -m "npm publish"
### 更新package版本号
npm version patch
### 推送至github触发action
git push

## 可用URL

### jsDelivr+github链接
https://cdn.jsdelivr.net/gh/[GithubUserName]/[AssetsRepo]/img/index.png

### jsDelivr+npm链接
